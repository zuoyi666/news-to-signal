# Reports

This directory is intended for experiment reports generated by each run.

Recommended file structure:

- `artifacts/runs/exp_YYYYMMDD_name/summary.md` (per-run summary)
- `artifacts/runs/exp_YYYYMMDD_name/run_config.yaml` (per-run reproducibility snapshot)
- `artifacts/runs/exp_YYYYMMDD_name/metrics.json` (per-run metrics payload)
- `artifacts/runs/exp_YYYYMMDD_name/signals_manifest.json` (signal schema manifest)

Keep this folder for cross-run narrative docs, decision logs, and benchmark summaries.
